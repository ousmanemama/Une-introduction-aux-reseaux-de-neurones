# Une introduction aux réseaux de neurones
Ce projet utilise un modèle CNN pour classifier 3 types de maladies des feuilles de riz.

###  **Fichier requirements.txt :**
```txt
tensorflow>=2.8.0
opencv-python
matplotlib
seaborn
scikit-learn
tqdm
pandas
numpy
```
## Installation
```bash
pip install -r requirements.txt
```
